Bienvenue sur le Memory des chiens !

Venez jouer et faites le meilleur score possible


Règle du jeu :

  • Retourner deux cartes
  • Si les images sont identiques, le jeu laisse les cartes visibles
  • Si les images sont différentes, les cartes retournent face cachée
  • La partie est terminée lorsque toutes les cartes ont été assemblées en paires


Si vous voulez vous souvenir de votre score, inscrivez-vous via le formulaire
Le jeu garde les 5 meilleurs scores
Appuyez sur la barre d'espace pour relancer une partie
